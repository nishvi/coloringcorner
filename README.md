Kids Cartoon Coloring Corner

Welcome to the Kids Cartoon Coloring Corner! This is a simple, fun, and interactive web application designed for children to explore their creativity through coloring. This project serves as a companion to our YouTube channel, providing free, printable coloring pages for our community.

Features

50 Unique Shapes: A wide variety of characters including Animals, Fruits, Space, Nature, and Basic Shapes.

Print-Ready: Optimized for printing directly from the browser with high-quality SVG outlines.

Coloring Inspiration: A dedicated section showing colorful examples to spark children's imagination.

Fast and Lightweight: A single-file HTML/JavaScript application that loads instantly.

Responsive Design: Works perfectly on desktops, tablets, and mobile phones.

Category Filters: Easily find specific types of coloring pages.

How to Use

Browse the gallery or use the search bar to find a character.

View the coloring inspiration section for ideas.

Click the "Print Coloring Page" button on any character card.

Color it in and share your masterpiece with us!

Tech Stack

HTML5 & CSS3

Tailwind CSS (Styling)

Lucide Icons (Visuals)

Vanilla JavaScript (Functionality)
